Die Anwendungen nutzen folgende moderne JavaScript-Features:
- ES6+ Syntax (Arrow Functions, Template Literals)

- CSS Custom Properties (CSS Variables)
- CSS Grid und Flexbox

## Datenschutz & Sicherheit

### Datenverarbeitung
- ✅ Alle Daten bleiben im Browser
- ✅ Keine Cookies
- ✅ Kein localStorage (Standard-Version)
- ✅ Keine Analytics/Tracking
- ✅ Keine externe API-Calls (außer CDN-Ressourcen)

### Content Security Policy
Empfohlene CSP für Produktiv-Deployment:
```
Content-Security-Policy: 
  default-src 'self'; 
  script-src 'self' https://cdnjs.cloudflare.com 'unsafe-inline'; 
  style-src 'self' https://fonts.googleapis.com 'unsafe-inline'; 
  font-src 'self' https://fonts.gstatic.com;
  img-src 'self' data:;
```
